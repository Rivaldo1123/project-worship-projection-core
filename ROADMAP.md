# Roadmap

## 0.1 — Projection state foundation

- Deterministic cue navigation
- Live, black, logo, and freeze output modes
- Safe display-selection policy
- Runtime cue validation
- Event-driven controller
- Strict TypeScript, tests, coverage, and CI

## 0.2 — Timing and overlays

- Countdown and elapsed-time state machines
- Non-destructive overlay model
- Alert-banner lifecycle
- Ticker state and priority rules
- Clock abstraction for deterministic tests

## 0.3 — Portable service plans

- Versioned JSON schema
- Validation diagnostics with precise paths
- Migration helpers between schema versions
- Song-section and sermon-slide cue helpers
- Import/export examples without licensed content

## 0.4 — Host adapters

- Reference Electron display adapter
- Reference React bindings
- Serialization and state-recovery helpers
- Keyboard-command mapping
- Accessibility guidance for operator interfaces

## 1.0 — Stable public API

- Documented compatibility policy
- Full API reference
- Integration example application
- Performance and reliability benchmarks
- Independent adopter feedback incorporated

## Future exploration

- Collaborative service-plan format
- Plugin-safe cue render contracts
- Internationalization helpers
- Remote-control protocol specification
- Tauri and browser presentation adapters

Roadmap priorities may change based on contributor and adopter feedback. Issues labeled `help wanted` and `good first issue` will identify approachable work.
